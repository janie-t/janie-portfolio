HTML, CSS and the DOM: Part 1

My kids were colouring in a blank poster today, with just the outlines of some animal pictures. HTML is like that basic outline. It provides the content of text, images, video etc, but nothing else. CSS is like the animals being coloured in different hues, and also given spots or stripes. CSS provides the styling such as colour and font. The DOM is the structure of the poster. It is a way of interacting with the HTML and CSS and provides a familiar structure that helps us to find familiarity in every website.
